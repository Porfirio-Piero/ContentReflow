# ContentReflow

> Know where every piece of content has been.

Dead-simple content repurposing tracker for solo creators. See which blog posts have become tweets, which newsletters turned into LinkedIn articles, and what content is still untouched.

## Problem

Content creators with multi-platform presence can't track which pieces have been repurposed where. They waste time recreating content they already have and miss repurposing opportunities.

## Features

- 📊 **Content Inventory + Status Matrix** — See all your content with a visual grid showing which platforms each piece has hit
- 📝 **Detail View with Repurposing Notes** — Click any piece to see what was changed, new URLs, and notes per platform
- 🔍 **Dashboard with Gap Analysis** — Find untouched content and neglected platforms at a glance

## Tech Stack

- Next.js 14 (App Router)
- Tailwind CSS
- shadcn/ui
- localStorage (no server needed)
- Vercel deployment
